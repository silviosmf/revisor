**Project Reviewer Description**

The Reviewer project is a tool to support systematic literature reviews, designed to classify and consolidate articles in accordance with the PRISMA protocol. It organizes bibliographic databases, processes batches of review prompts, and generates sets of relevant articles for qualitative and quantitative analysis.

Role in the review project
Performs automatic extraction and classification of articles by thematic intersection.
Generates structured results in JSON and CSV files, facilitating screening and validation.
Produces final outputs compatible with the PRISMA workflow, including selection of relevant articles and documentation of the consolidated database.

Application in the specific case
In this project, the Reviewer supports the review of the article: AI Agents in Service Recovery: An Agency Theory Framework for Governance, Capabilities, and B2B Engagement
